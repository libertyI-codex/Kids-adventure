# こどもの冒険

## 概要

「こどもの冒険」は、子どもの毎日の頑張りが冒険の世界へ反映される生活習慣アプリです。

Ver.1.0 試作5の子ども向け表示名は **結羽ちゃんの冒険** です。

## バージョン

- 表示名: 結羽ちゃんの冒険
- 内部プロジェクト名: こどもの冒険
- バージョン: Ver.1.0 試作5
- appVersion: `1.0.0-prototype.5`
- schemaVersion: `1`

## 基本体験

1. 今日のおしごとをする
2. スターをもらう
3. ぬりえを解放する
4. 自由に色を塗る
5. まほうの仕上げで作品が変身する
6. 完成した作品が森に登場する
7. 今日のぼうけんと思い出が残る
8. あつめたほし10こごとに、ふしぎなたまごが増える

## 対象年齢

4〜8歳を想定しています。

## 技術条件

完全無料で使える静的Webアプリです。

- HTML
- CSS
- JavaScript
- SVG
- localStorage

外部API、外部AI、有料サービス、npm依存、ビルドツール、CDN、ログイン、クラウド保存は使っていません。

`index.html` をブラウザで直接開いても動作します。GitHub Pagesにも対応します。

## 初期おしごと

- はみがき
- おきがえ
- えほん
- ピアノ
- ドリル
- そろばん

各おしごとの初期報酬は1スターです。1つのおしごとは1日1回だけスターを獲得できます。

既存データに旧おしごとの `おみずあげ` がある場合は削除しません。試作2の補完処理では、新しい初期おしごとの不足分だけを追加します。

## 初期ぬりえと解放順

- ちょうちょ: 4スター
- おはな: 8スター
- うさぎ: 12スター
- ねこ: 16スター
- イルカ: 20スター
- きょうりゅう: 24スター
- うま: 28スター

作品はBase64画像として保存せず、SVGテンプレートIDと領域別カラー情報から再描画します。試作5では蝶以外の6種類を旧SVGパスを流用せず全面的に描き直し、未着色の小さなプレビューでも種類を判別しやすい輪郭にしました。

組み込みぬりえには `designVersion`、`svgKey`、`viewBox`、領域定義、旧領域ID対応を持たせています。起動時の `ensureDataShape()` でコード内の正式定義を保存済みテンプレートへ同期し、解放済み状態や作品、森の配置、親コメントなどの進行状況は維持します。

旧作品の `regionColors` は、新旧領域IDの対応表に基づいて新領域へ補完します。既存作品は削除せず、アルバムと森に残ります。

## ふしぎなたまご

累計スター10こごとに、たまごを1こ獲得します。つかえるスターの残高やぬりえ解放には影響しません。

Ver.1.0 試作5では、たまごの状態は `new` のみ使用します。将来の拡張用に `cracked`、`hatched` を追加できる構造にしています。

旧データには `ensureDataShape()` 相当の補完処理で `eggInventory` を追加します。累計スターに応じた不足分だけを追加し、重複追加はしません。

## 音

外部音源は使用せず、Web Audio APIで生成します。

- BGM: 森の雰囲気に合う短いループ。初期値はOFFです。
- 効果音: スター獲得、ぬりえ完成、たまご獲得、生き物誕生用を用意しています。
- 親モードからBGMと効果音を個別にON/OFFできます。

## Safari長押し改善

親モードの「おとな」ボタンは、iPhone Safariで文字選択が出にくいように調整しています。

- `user-select: none`
- `-webkit-user-select: none`
- `-webkit-touch-callout: none`
- `touch-action: manipulation`
- `pointerdown` / `pointerup` / `pointercancel` による長押し判定

## 保存キー

- アプリデータ: `kodomoAdventure.appData.v1`
- UI状態: `kodomoAdventure.uiState.v1`
- 一時バックアップ: `kodomoAdventure.backup.v1`

試作5でも保存キーは変更していません。`templateId` と `regionColors` による作品保存方式も維持しています。

## UI調整

試作5では画面全体の過剰な丸みを調整しました。カード、パネル、ボタン、入力欄、モーダルの角丸を用途別に分け、子ども向けのやさしさは残しながら画面構造を読み取りやすくしています。

## テスト用ページ

`tests/coloring-preview.html` で全7種類のぬりえを一覧確認できます。

- 未着色状態
- サンプル着色状態
- 120pxプレビュー
- 大きなプレビュー
- designVersion
- 領域ID一覧

このページは本番画面からリンクしていません。GitHub Pages公開用フォルダにもコピーしません。

## データバックアップ方法

親モードからデータ管理を開き、JSONを書き出せます。復元時はJSON形式、必須項目、schemaVersionを確認し、不正データなら現在のデータを変更しません。
