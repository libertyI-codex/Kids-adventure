# こどもの冒険 Ver.1.0 試作6 実装メモ

## 概要

試作6では、指定されたぬりえ原画の判別性を上げ、森に登場した作品を利用者が自由に配置変更できるようにした。

- appVersion: `1.0.0-prototype.6`
- schemaVersion: `1`
- 保存キー: 変更なし
- 画像生成AI、外部画像、外部SVG、素材ダウンロード: 未使用

## 原画の修正方針

### おはな

中心、茎、左右の葉は試作5のSVG形状と領域IDを維持した。花弁だけを描き直し、丸い団子の集合ではなく、中心へ向かう涙形の花びらとして判別しやすくした。

### うさぎ

横向きの全身シルエットへ修正した。長い左右の耳、前へ出た鼻先、胴体、大きく曲がった後脚、独立した前脚、丸いしっぽを強調した。

### きょうりゅう

ティラノサウルス風の横向きシルエットへ修正した。大きく前へ伸びた頭、下あご、首、胴体、小さな前脚、太い後脚、長いしっぽを分けて描いた。

### うま

横向きに立つ馬へ修正した。縦長の顔、左右の耳、首、たてがみ、胴体、4本の脚、ひづめ、長いしっぽを分けた。

### イルカ

横向きに泳ぐ流線形へ修正した。口先、背びれ、胸びれ、腹側、細い尾の付け根、上下に分かれた尾びれを明確にした。

## 旧領域IDとの対応

領域IDは原則として試作5から維持した。既存作品の `regionColors` はそのまま新しい原画へ適用できる。

`regionAliases` も維持しているため、さらに古い作品で使われた旧IDも可能な範囲で補完される。

おはなは中心、茎、葉のIDとパスを維持した。

- `center`
- `stem`
- `leaf_left`
- `leaf_right`

## 配置編集モード

森画面に「ならべかえ」ボタンを追加した。

通常モード:

- 作品タップで詳細を表示
- ドラッグ移動は無効

配置編集モード:

- 「すきなところへ うごかしてね」を表示
- 「できた」で配置を確定保存
- 「もとにもどす」で自動配置へ戻す
- 「やめる」で編集前の配置へ戻す
- 作品詳細モーダルは開かない
- 作品アニメーションを一時停止

## 座標保存方式

作品位置は森の表示領域に対する割合で保存する。

```json
{
  "xPercent": 52.4,
  "yPercent": 68.1,
  "zIndex": 4,
  "isManual": true,
  "updatedAt": "2026-07-14T..."
}
```

互換性のため、既存の `x` / `y` も `xPercent` / `yPercent` と同じ値へ同期する。

## 既存自動配置からの移行

既存作品に `xPercent` / `yPercent` がない場合は、初回の `KA.worlds.getPlacements()` で既存の `x` / `y` から一度だけ補完する。

手動配置済みの作品は `isManual: true` とし、自動配置処理で上書きしない。

「もとにもどす」を実行した場合のみ、固定seedを使って自動配置を再計算し、`isManual: false` に戻す。

## zIndex処理

ドラッグした作品は一時的に最前面へ表示する。確定時には `normalizeZIndexes()` で前後関係を1から連番へ正規化し、極端に大きな値が増え続けないようにした。

## iPhone Safariのタッチ対策

配置編集モード中の作品にだけ以下を適用する。

- `user-select: none`
- `-webkit-user-select: none`
- `-webkit-touch-callout: none`
- `touch-action: none`

通常時の森画面全体のスクロールは常時禁止しない。

## 互換性対応

- 保存キーは変更していない。
- schemaVersionは `1` のまま。
- 既存作品、親コメント、お気に入り、森配置、スター履歴、たまごは削除しない。
- 解放済みぬりえは再ロックしない。
- `templateId` と `regionColors` による保存方式を維持した。
