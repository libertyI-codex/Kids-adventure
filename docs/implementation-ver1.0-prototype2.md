# 実装メモ Ver.1.0 試作2

## 変更内容

Ver.1.0 試作2では、初期おしごとと初期ぬりえを拡張した。

## 初期おしごと

新規初期データでは次の6項目を作成する。

- はみがき
- おきがえ
- えほん
- ピアノ
- ドリル
- そろばん

各おしごとの初期報酬は1スター。

既存データでは `ensureDataShape()` により不足している新規初期おしごとだけを追加する。既存の `task_watering` は削除しない。

## 初期ぬりえ

次の2種類を追加した。

- イルカ
- きょうりゅう

どちらもJavaScript内のSVGテンプレートとして実装し、他のぬりえと同じく `templateId` と `regionColors` で保存・再描画する。

## 解放順

- ちょうちょ: 4スター
- おはな: 8スター
- うさぎ: 12スター
- ねこ: 16スター
- イルカ: 20スター
- きょうりゅう: 24スター

既存データの組み込みぬりえテンプレートは、IDを基準に必要スター・表示順・領域定義を同期する。解放済みぬりえは再ロックしない。

## 互換性

- schemaVersionは1のまま。
- appVersionは `1.0.0-prototype.2`。
- 保存キーは変更しない。
- 作品保存方式は変更しない。
- 旧データにある作品、スター履歴、日別記録、解放状態は削除しない。
